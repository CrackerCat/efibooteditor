# EFI Boot Editor

Boot Editor for (U)EFI based systems.

![EFIBootEditor](doc/efibooteditor.png)
![File path dialog](doc/filepathdialog.png)

## Requirements

* [Qt](//www.qt.io/) (>=5.15.2)
* [efivar](//github.com/rhboot/efivar) (>=37) on linux
